rlairgo - powered by Echo

使用说明

GET

video 获取视频列表
video/:id 获取视频详细信息
comment/:id 获取视频评论

POST

video 新增一条视频
数据格式:
{
"title": "坚果Pro",
"imageUrl": "http://rlair.live/image.jpg",
"src": "http://rlair.live/demo.mp4",
"videoId": "10001",
"introduction": "坚果 Pro，是锤子科技在2017年5月9日晚在深圳正式发布的旗下的第五款新机。",
"source": "smartisan",
"timeStamp": "1501150459"
}

comment 新增一条评论
数据格式:
{
"videoId": "10001",
"username": "Echo",
"headImageUrl": "http://rlair.live/headImage.jpg",
"context": "真棒",
"timeStamp": "1501150459"
}

PUT

video/:id 更新视频信息
数据格式:
{
"title": "修改标题"
}

DELETE

video/:id 删除视频

备注

PUT、POST 需要在 headers 里加入 Content-Type: application/json
